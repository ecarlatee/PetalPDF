# PetalPDF 🌸

PetalPDF est un assistant interactif en ligne de commande (CLI) pour télécharger automatiquement tous les fichiers PDF présents sur une page web. Il intègre une jolie interface bilingue (Français/Anglais), des animations et un magnifique ASCII art.

![Bannière](https://img.shields.io/badge/Python-3.6%2B-blue)
![Licence](https://img.shields.io/badge/License-MIT-green)

## Fonctionnalités ✨

- **Mode Assistant (Wizard)** : Plus besoin de retenir des lignes de commande complexes, le programme vous pose des questions.
- **Bilingue (FR/EN)** : Choisissez votre langue au démarrage.
- **Esthétique** : Animations fluides, codes couleurs clairs et design chaleureux.
- **Protection Anti-doublon** : Le programme détecte si un fichier est déjà téléchargé pour éviter de perdre du temps.

## Installation 🛠️

1. Clonez ce dépôt GitHub sur votre machine :
   ```bash
   git clone https://github.com/votre-nom-utilisateur/PetalPDF.git
   cd PetalPDF
   ```

2. Installez les librairies requises (via `pip`) :
   ```bash
   pip install -r requirements.txt
   ```

## Utilisation 🚀

Lancez simplement le script avec Python :
```bash
python petalpdf.py
```

Laissez-vous guider par l'interface ! PetalPDF vous demandera :
1. La langue (fr ou en)
2. L'URL de la page web que vous souhaitez analyser
3. Le nom du dossier dans lequel sauvegarder les PDF trouvés.

## Avertissement
Assurez-vous de posséder les droits nécessaires pour télécharger les fichiers sur les sites ciblés.
